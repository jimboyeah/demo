401	1	2013-08-18 18:07:51	2013-08-18 10:07:51	<h1>组织人事处职责</h1>\
<p>负责拟订学校党建工作计划并组织实施；负责学校基层组织建设及党员教育管理与发展工作；负责学校统战工作；负责学校党内统计和党费管理工作。负责拟订 学校人事工作计划并组织实施；负责学校的人事管理、机构编制和劳动工资等工作；负责学校干部培训工作；参与学校人事制度改革。</p>\
[caption id="attachment_70" align="alignnone" width="300"]<a href="http://www.wp.com/HR/wp-content/uploads/2013/08/014.jpg"><img class="size-medium wp-image-70" alt="行政楼" src="http://www.wp.com/HR/wp-content/uploads/2013/08/014-300x168.jpg" width="300" height="168" /></a> 行政楼[/caption]\
<p align="left"><strong>处长职责分工 </strong><br /> <strong>概要：</strong>主持组织人事处全面工作，负责学校师德师风建设，直管教职员工年度考核工作。<strong> </strong><br /> <strong>具体： </strong></p>\
<ol>\
<li>负责组织制定学校基层组织建设、师德师风建设、机构设置、人员编制管理、人员考核评价等的规划与方案；</li>\
<li>主持制订部门工作计划，并负责抓好组织实施及总结工作；</li>\
<li>负责对部门工作人员进行思想政治教育、组织管理和考核考评；</li>\
<li>负责做好引进人才、干部政策和知识分子政策落实工作；</li>\
<li>负责组织对调进人员（或聘用人员）的考察审查工作；</li>\
<li>负责学校的工资、津贴、加班费、奖酬金的审批工作；</li>\
<li>主持学校各类人员专业技术职务评聘及教师资格认定工作；</li>\
<li>负责学校副科级以上干部的考察、考核工作，后备干部的推荐选拔工作，向党委提出培养、调配、使用意见；</li>\
<li>负责部门内外，学校内外的组织人事协调工作；</li>\
<li>主持部门办公会议，审批部门办公经费；</li>\
<li>完成学校领导和上级有关部门交办的其他工作。</li>\
</ol>\
<p align="left"><strong>副处长(1)</strong><br /> <strong>概要：</strong>分管党建、组织、统战、人事、及职教能力测评等工作,直管组织人事科。<br /> <strong>具体： </strong></p>\
<ol>\
<li>负责起草部门计划、汇报、总结、规章制度等；</li>\
<li>负责起草党建、组织、统战、人事、编制等方面的计划、汇报、总结、规章制度等；<strong> </strong></li>\
<li>负责抓好党建、组织、统战、人事、编制等工作的落实；</li>\
<li>协助处长制定学校基层组织建设、机构设置、人员编制管理等的规划与方案；</li>\
<li>协助处长做好引进人才、干部政策和知识分子政策落实工作；</li>\
<li>负责做好调进人员（或聘用人员）的考察审查工作；</li>\
<li>完成学校领导和上级有关部门交办的其他工作。<strong> </strong></li>\
</ol>\
<p align="left"><strong>副处长(2)</strong><br /> <strong>概要：</strong>分管劳资、师资培训、职称评审等有关工作, 协管师德师风建设，直管劳资科、师资科。<br /> <strong>具体： </strong></p>\
<ol>\
<li>负责起草劳资、师资培训、职称评审、职教能力测评、退离休教职工等方面的计划、汇报、总结、规章制度等；</li>\
<li>负责抓好劳资、师资培训、职称评审、职教能力测评、退离休教职工等工作的落实；<strong></strong></li>\
<li>协助处长制定学校师德师风建设、绩效管理、人员考核评价等的规划与方案；</li>\
<li>负责做好学校的工资、津贴、加班费、奖酬金的审核工作；</li>\
<li>负责做好学校各类人员专业技术职务评聘及教师资格认定工作；</li>\
<li>兼任部门发言人，负责撰写并报送部门新闻（信息）、简报等材料；<strong> </strong></li>\
<li>完成学校领导和上级有关部门交办的其他工作。</li>\
</ol>\
<p align="left"><strong>组织人事科 </strong></p>\
<ol>\
<li>做好党务日常工作；</li>\
<li>做好人员进出与调配、教职工退休、离职等工作；</li>\
<li>做好各类人事信息数据的统计及上报工作；</li>\
<li>管理职工人事档案，管理人事处网页；</li>\
<li>承担教职工考核奖惩有关工作；</li>\
<li>完成学校领导和上级有关部门交办的其他工作。<strong> </strong></li>\
</ol>\
<p align="left"><strong>劳资科 </strong></p>\
<ol>\
<li>管理教职工工资，办理教职工工资晋级、变动、关系转移，新教工转正定级等手续；</li>\
<li>做好教职工工资、绩效考核津贴、奖金及福利的汇总制表工作；</li>\
<li>做好劳资等相关信息的统计、填报工作；</li>\
<li>做好教职工请销假管理，探亲路费核定等工作；</li>\
<li>承担教职工考核奖惩有关工作；</li>\
<li>完成学校领导和上级有关部门交办的其他工作。<strong> </strong></li>\
</ol>\
<p align="left"><strong>师资科 </strong></p>\
<ol>\
<li>做好教职工进修、培训及进企业实践锻炼的有关管理工作；</li>\
<li>做好教师资格认定、教职工专业技术职务及技能等级的评聘工作；</li>\
<li>做好教师专业技术水平提升工程及 “千百十工程”培养人选的管理工作；</li>\
<li>做好师资培训、职称评审、岗位聘任、职教能力测评等工作；</li>\
<li>承担教职工考核奖惩有关工作；</li>\
<li>完成学校领导和上级有关部门交办的其他工作。</li>\
</ol>\

409	1	2013-08-15 23:53:59	2013-08-15 15:53:59	<div align="center"><b>河源职业技术学院兼职教师管理办法</b></div>\
<div></div>\
<div>    为适应我院办学需要，进一步改善教师队伍结构，提高办学效益，充分发挥兼职教师在教学科研工作中的作用，建立一支教学水平高、实践经验丰富、结构合理的兼职教师队伍，特制定本办法。</div>\
<div>    一、聘用对象：</div>\
<div>    1.在高校或其他学校任职的名师专家；</div>\
<div>    2.在企事业及社会中具有丰富实践经验的高级管理人员、技术骨干或能工巧匠等；</div>\
<div>    3.返聘的退休教师。</div>\
<div>    二、兼职教师聘用条件：\
1.遵纪守法，热爱教育事业，具有良好的思想品德和职业道德。</div>\
<div>    2.身体健康，具有良好的语言表达能力，能胜任专业课程的教学或实习实训指导工作，能保质保量地完成学院交给的各教学环节的工作任务。</div>\
<div>    3.兼职专业理论课教师必须具有中级及以上专业技术职务；兼职实习实训指导教师原则上应具有中级及以上专业技术职务或高级技工及以上技能等级证书，实践经验丰富，实践技能特别强者可适当放宽职称要求，但专业结构应与学校专业设置相适应，且为学院紧缺或急需人才。\
4.返聘退休教师的年龄，男不超过65周岁，女不超过60周岁，身体健康。\
三、兼职教师职责：\
1.担任校内课程教学任务或举办讲座，或承担指导学生实习实训任务。</div>\
<div>    2.严格执行教学进度，按照教学大纲和计划完成规定的教学工作量。\
3.突出高职教育特点，培养学生“手、脑并用”，培养学生的专业实践操作能力，教书育人。</div>\
<div>    四、聘用程序：\
1.校内课程教学兼职教师：</div>\
<div>    (1)系部根据学科发展和专业建设、实践教学的需要，将招聘计划及对应聘人员的具体要求书面报人事处。</div>\
<div>    ⑵人事处征求教务处意见，根据教学工作需要及师资队伍现状，对计划提出初审意见，报学院领导审批。</div>\
<div>    ⑶人事处发布招聘信息；会同系部组织对应聘教师进行教学能力的面试考核；与初选出的拟聘对象商谈工作任务、聘用方式、待遇、聘期等具体事项。</div>\
<div>    ⑷人事处提出聘用方案，报学院批准。</div>\
<div>    ⑸人事处与受聘人签订聘用协议书，相关系部拟定好应聘人员具体的工作职责与要求，作为附件。</div>\
<div>    ⑹兼职教师的续聘：在聘期结束前两个月，由所在系部就下一阶段聘任工作提出意见，如经考核合格可以续聘，经兼职教师本人同意，办理续聘手续，签订聘任协议。</div>\
<div>    2.校外实习实训基地兼职教师：</div>\
<div>由系部与实习实训基地的领导或人事部门商定，报学院人事处。</div>\
<div>    五、校内课程教学兼职教师的管理：\
1.兼职教师由所在系部进行管理。有关系部应向兼职教师介绍情况，明确其担负的教学任务及教学要求，并提供相关的资料和工作条件。\
2.教务处和系部要定期组织学生座谈会，听取学生对兼职教师教学效果、教学质量的意见，并及时反馈给教师本人。兼职教师接受学院组织的期中、期末教学评估考核工作。\
3.兼职教师在工作中如出现教学态度较差、教学质量不能保证等情况，经教务部门评估，并经指出仍无改进者，应予以解聘。</div>\
<div>    4.兼职教师因各种原因不能履行职责或工作中发生严重失职造成不良影响的，经查属实者，应予以解聘。</div>\
<div>    5.兼职教师因事请假应履行请假手续，原则上必须在事前两天通知系部和教务处，以便及时调课或安排补课。未经请假而未上课者，按缺课节数扣发两倍的课酬。\
6.期末由系部与教务处根据考核情况评选出优秀兼职教师，根据实际情况给予适当的奖励，并写出书面材料，报人事处备案，作为今后续聘的主要依据 。</div>\
<div>    7.聘请外籍教师按省外事办规定办理。</div>\
<div>    六、兼职教师聘用待遇：</div>\
<div>校内课程教学兼职教师聘用待遇按《河源职业技术学院聘用机动编制人员实施办法》（河职院［2006］30号）。校外实习实训基地兼职教师聘用待遇根据实际另行商定。</div>\
<div>    七、本办法自发文之日起执行，未尽事宜由人事处负责解释。</div>\

<p><a id="Description" name="Description"></a></p>\
<h2>Description</h2>\
<p>Create or modify a <a title="Glossary" href="http://codex.wordpress.org/Glossary#Post_Type">post type</a>. <strong>register_post_type</strong> should only be invoked through the <tt><a title="Plugin API/Action Reference/init" href="http://codex.wordpress.org/Plugin_API/Action_Reference/init"> 'init'</a></tt> action. It won't work at all if called before <tt>'init'</tt>, and aspects of the new post type will work incorrectly if called later.</p>\
<p><a id="Taxonomies" name="Taxonomies"></a></p>\
<h2>Taxonomies</h2>\
<p>It's important to note, that you should in every way, always register your taxonomies using the <code>taxonomies</code> argument when you register your post type. Else they won't get recognized as connected when using filters like <code>parse_query</code> or <code>pre_get_posts</code>, which can lead to unexpected results and failure.</p>\
<p><a id="Reserved_Post_Types" name="Reserved_Post_Types"></a></p>\

<li>post</li>\
<li>page</li>\
<li>attachment</li>\
<li>revision</li>\

<h2>Flushing Rewrite on Activation</h2>\
<p>To get permalinks to work when you activate the plugin use the following example, paying attention to how my_cpt_init is called in the register_activation_hook callback:</p>\
<pre>add_action( 'init', 'my_cpt_init' );\
function my_cpt_init() {\
    register_post_type( ... );\
}\
\
function my_rewrite_flush() {\
    // First, we "add" the custom post type via the above written function.\
    // Note: "add" is written with quotes, as CPTs don't get added to the DB,\
    // They are only referenced in the post_type column with a post entry, \
    // when you add a post of this CPT.\
    my_cpt_init();\
\
    // ATTENTION: This is *only* done during plugin activation hook in this example!\
    // You should *NEVER EVER* do this on every page load!!\
    flush_rewrite_rules();\
}\
register_activation_hook( __FILE__, 'my_rewrite_flush' );\
</pre>\
<p>For themes, you'll need to use the after_switch_theme hook instead. Like so:</p>\
<pre>add_action( 'init', 'my_cpt_init' );\
function my_cpt_init() {\
    register_post_type( ... );\
}\
\
function my_rewrite_flush() {\
    flush_rewrite_rules();\
}\
add_action( 'after_switch_theme', 'my_rewrite_flush' );\
</pre>\

<h4>1.什么是自定义文章类型</h4>\
<p><a title="查看 WordPress自定义文章类型 中的全部文章" href="http://www.wpyou.com/tag/wordpress%e8%87%aa%e5%ae%9a%e4%b9%89%e6%96%87%e7%ab%a0%e7%b1%bb%e5%9e%8b" target="_blank">WordPress自定义文章类型</a>是WordPress自3.0版本以后新增的一个非常实用的特色功能。WordPress内置的两种文章类型：<strong>Post</strong>（文章）和<strong> Page</strong>（页面），Post一般作为经常更新的文章使用（如博客日志），Page一般作为静态页面使用（如网站的关于和联系页面）。那么使用<a title="查看 WordPress自定义文章类型 中的全部文章" href="http://www.wpyou.com/tag/wordpress%e8%87%aa%e5%ae%9a%e4%b9%89%e6%96%87%e7%ab%a0%e7%b1%bb%e5%9e%8b" target="_blank">WordPress自定义文章类型</a>就可以实现更多除Post、Page等以外的更多自定义文章类型，而且你可以设置各不同文章类型的诸多属性。</p>\
<h4>2.如何创建自定义文章类型</h4>\
<p><strong>2,1 通过自定义文章类型插件</strong><a href="http://wordpress.org/extend/plugins/custom-post-type-ui/" target="_blank"><strong>Custom Post Type UI</strong></a><strong>实现</strong></p>\
<p>使用<a href="http://wordpress.org/extend/plugins/custom-post-type-ui/" target="_blank">Custom Post Type UI</a>自定义文章类型插件可以非常方便地对新增的文章类型进行管理设置，设置界面如下图：<a href="http://pic.yupoo.com/housne/480249c08188/63hf4c2z.png">http://pic.yupoo.com/housne/480249c08188/63hf4c2z.png</a></p>\
<p><strong>2,2 手工增加自定义文章类型功能</strong></p>\
<p>在Wordpress官方有非常详细的说明文档，使用可以多参考下官方介绍 <a href="http://codex.wordpress.org/Custom_Post_Types">Custom Post Types</a> 或是 <a href="http://justintadlock.com/archives/2010/04/29/custom-post-types-in-wordpress/">Custom post types in WordPress</a>。</p>\
<p>本文以创建（Event）文章类型为例，记录一些事件比如个人生日、假日、会议日期等。</p>\
<p>2,2,1 首先，你需要在主题包的functions.php文件中增加以下代码段来使得主题支持该功能：</p>\
<blockquote>\
<p>add_action( ‘init’, ‘create_events’ );<br /> function create_events() {<br /> $labels = array(<br /> ‘name’ =&gt; _x(‘Events’, ‘post type general name’),<br /> ‘singular_name’ =&gt; _x(‘Event’, ‘post type singular name’),<br /> ‘add_new’ =&gt; _x(‘Add New’, ‘Event’),<br /> ‘add_new_item’ =&gt; __(‘Add New Event’),<br /> ‘edit_item’ =&gt; __(‘Edit Event’),<br /> ‘new_item’ =&gt; __(‘New Event’),<br /> ‘view_item’ =&gt; __(‘View Event’),<br /> ‘search_items’ =&gt; __(‘Search Events’),<br /> ‘not_found’ =&gt; __(‘No Events found’),<br /> ‘not_found_in_trash’ =&gt; __(‘No Events found in Trash’),<br /> ‘parent_item_colon’ =&gt; ”<br /> );<br /> $supports = array(‘title’, ‘editor’, ‘custom-fields’, ‘revisions’, ‘excerpt’);<br /> register_post_type( ‘event’,<br /> array(<br /> ‘labels’ =&gt; $labels,<br /> ‘public’ =&gt; true,<br /> ‘supports’ =&gt; $supports<br /> )<br /> );</p>\
<p>}</p>\
</blockquote>\
<p><strong>add_action</strong>告诉wordpress初始化时调用函数<strong>create_events</strong></p>\
<p><strong>$label</strong>数组告诉wordpress如何显示这个文章类型的相关信息</p>\
<p><strong>$supports</strong>数组告诉wordpress这个文章类型可以支持什么（比如文章摘要excerpt）</p>\
<p><strong>register_post_type</strong>就是在wordpress注册这个新的文章类型，这个函数有很多选项可用，具体可以参阅相关文章。</p>\
<p>一旦我们把代码添加到function.php后，我们可以在后台管理页面看到如下页面：</p>\
<p><img title="06-02_wordpress_custom_post_type_registered" alt="wordpress-自定义文章类型" src="http://www.wpyou.com/wp-content/uploads/2011/07/092329BPL.png" width="549" height="284" /></p>\

<p><img title="06-03_wordpress_custom_post_type_publish_new_type" alt="wordpress-自定义文章类型" src="http://www.wpyou.com/wp-content/uploads/2011/07/092331QTD.png" width="550" height="413" /></p>\
<h4>3.显示自定义文章类型</h4>\
<p>添加完事件之后发布，然后编辑事件，点击View Event（查看事件），你就可以看到事件就像正常文章一样显示了。</p>\
<p><img title="06-04_wordpress_custom_post_type_just_like_post" alt="wordpress-自定义文章类型" src="http://www.wpyou.com/wp-content/uploads/2011/07/0923327VC.jpg" width="550" height="451" /></p>\
<p>当 然我们在wordpress自定义文章类型是因为它们和一般的文章是不一样的。wordpress内部有一个很好的机制用来自定义文章类型的外观，如果你 曾经做过wordpress主题，应该知道，一般文章（post）对应的模板文件是single.php。wordpress3.0允许你用自定义的模板 文件（如single-event.php）来定制我们新的文章类型的外观。</p>\
<p>为了简便起见，我们将single.php复制并命名为single-event.php，为了演示看是否生效，我们对代码做一些改动。</p>\
<p><strong>原代码</strong>：</p>\
<blockquote>\
<p><code>&lt;</code><code>h1</code> <code>class</code><code>=</code><code>"entry-title"</code><code>&gt;&lt;?</code><code>php</code> <code>the_title(); ?&gt;&lt;/</code><code>h1</code><code>&gt;</code></p>\
</blockquote>\
<p><strong>新代码</strong>：</p>\
<blockquote>\
<p><code>&lt;</code><code>h1</code> <code>class</code><code>=</code><code>"entry-title"</code><code>&gt;Event: &lt;?</code><code>php</code> <code>the_title(); ?&gt;&lt;/</code><code>h1</code><code>&gt;</code></p>\
</blockquote>\
<p>回到刚才的事件页面，我们发现标题已经发生了改变。有需要的话我们可以进一步修改single-event.php文件。</p>\
<p><img title="06-05_wordpress_custom_post_type_updated" alt="wordpress-自定义文章类型" src="http://www.wpyou.com/wp-content/uploads/2011/07/092341zjk.jpg" width="551" height="448" /></p>\
<h4>4.列出自定义文章类型</h4>\
<p>前面我们讲解如何创建、显示单个事件，但是如果我们需要像博客日志一样列出所有的事件应该如何操作？</p>\
<p>这里有很多方法可以实现这个功能，一个通常使用的方法是通过<a href="http://codex.wordpress.org/Pages#Page_Templates">Page Template</a>（页面模板）来实现，这个方法如果使用几次后发现它并不难，只是需要的步骤多一点。</p>\
<p>（1）首先复制一份page.php并将它命名为page-events.php。（可以在wordpress创建很多不同页面模板）。</p>\
<p>（2）在page-events.php头部添加一下代码，这些代码告诉wordpress这是一个模板文件，以及这个模板文件的 相关信息。</p>\
<blockquote>\
<p>&lt;?php<br /> /**<br /> * Template Name: Events Template<br /> * @package WordPress<br /> * @subpackage Twenty_Ten<br /> * @since Twenty Ten 1.0<br /> */<br /> get_header(); ?&gt;</p>\
</blockquote>\
<p>（3）先用下面的代码来获取所有的事件文章.</p>\
<blockquote>\
<p><code>&lt;?php query_posts(</code><code>array</code><code>(</code><code>'post_type'</code><code>=&gt;</code><code>'event'</code><code>)); ?&gt;</code></p>\
</blockquote>\
<p>（4）然后用wordpress loop的方法来显示出所有的事件文章。</p>\
<blockquote>\
<p><code>&lt;?php </code><code>if</code> <code>( have_posts() ) </code><code>while</code> <code>( have_posts() ) : the_post(); ?&gt;</code></p>\
</blockquote>\
<p>这 样我们就创建好了一个自定义的页面模板，接着创建一个Event页面来显示这些事件，进入wordpress后台的页面部分，创建一个新页面命名为 Event，然后在页面属性的模板选项里，选择我们刚才创建的模板文件即可。发布完后，到网站查看这个页面，可以发现所有的事件文章都列在上面了。根据不 同的主题，你可能需要修改导航栏的相关代码。</p>\
<p><img title="06-07_wordpress_custom_post_type_finished" alt="wordpress-自定义文章类型" src="http://www.wpyou.com/wp-content/uploads/2011/07/092342kAL.jpg" width="550" height="575" /></p>\
<p>通过修改page-events..php文件，我们可以自定义这个类型的文章的显示方式，比如为每个事件增加一个日历按钮或者只显示事件标题等。</p>\

<p>通常我们在创建完自定义文章类型后，就在后台新建一个页面，然后通过页面模版调用。分页的代码是出现在页面模版中，我们一般的调用代码是：</p>\

<h4>1.什么是自定义文章类型</h4>\
<p><a title="查看 WordPress自定义文章类型 中的全部文章" href="http://www.wpyou.com/tag/wordpress%e8%87%aa%e5%ae%9a%e4%b9%89%e6%96%87%e7%ab%a0%e7%b1%bb%e5%9e%8b" target="_blank">WordPress自定义文章类型</a>是WordPress自3.0版本以后新增的一个非常实用的特色功能。WordPress内置的两种文章类型：<strong>Post</strong>（文章）和<strong> Page</strong>（页面），Post一般作为经常更新的文章使用（如博客日志），Page一般作为静态页面使用（如网站的关于和联系页面）。那么使用<a title="查看 WordPress自定义文章类型 中的全部文章" href="http://www.wpyou.com/tag/wordpress%e8%87%aa%e5%ae%9a%e4%b9%89%e6%96%87%e7%ab%a0%e7%b1%bb%e5%9e%8b" target="_blank">WordPress自定义文章类型</a>就可以实现更多除Post、Page等以外的更多自定义文章类型，而且你可以设置各不同文章类型的诸多属性。</p>\
<h4>2.如何创建自定义文章类型</h4>\
<p><strong>2,1 通过自定义文章类型插件</strong><a href="http://wordpress.org/extend/plugins/custom-post-type-ui/" target="_blank"><strong>Custom Post Type UI</strong></a><strong>实现</strong></p>\
<p>使用<a href="http://wordpress.org/extend/plugins/custom-post-type-ui/" target="_blank">Custom Post Type UI</a>自定义文章类型插件可以非常方便地对新增的文章类型进行管理设置，设置界面如下图：<a href="http://pic.yupoo.com/housne/480249c08188/63hf4c2z.png">http://pic.yupoo.com/housne/480249c08188/63hf4c2z.png</a></p>\
<p><strong>2,2 手工增加自定义文章类型功能</strong></p>\
<p>在Wordpress官方有非常详细的说明文档，使用可以多参考下官方介绍 <a href="http://codex.wordpress.org/Custom_Post_Types">Custom Post Types</a> 或是 <a href="http://justintadlock.com/archives/2010/04/29/custom-post-types-in-wordpress/">Custom post types in WordPress</a>。</p>\
<p>本文以创建（Event）文章类型为例，记录一些事件比如个人生日、假日、会议日期等。</p>\
<p>2,2,1 首先，你需要在主题包的functions.php文件中增加以下代码段来使得主题支持该功能：</p>\
<blockquote>\
<p>add_action( ‘init’, ‘create_events’ );<br /> function create_events() {<br /> $labels = array(<br /> ‘name’ =&gt; _x(‘Events’, ‘post type general name’),<br /> ‘singular_name’ =&gt; _x(‘Event’, ‘post type singular name’),<br /> ‘add_new’ =&gt; _x(‘Add New’, ‘Event’),<br /> ‘add_new_item’ =&gt; __(‘Add New Event’),<br /> ‘edit_item’ =&gt; __(‘Edit Event’),<br /> ‘new_item’ =&gt; __(‘New Event’),<br /> ‘view_item’ =&gt; __(‘View Event’),<br /> ‘search_items’ =&gt; __(‘Search Events’),<br /> ‘not_found’ =&gt; __(‘No Events found’),<br /> ‘not_found_in_trash’ =&gt; __(‘No Events found in Trash’),<br /> ‘parent_item_colon’ =&gt; ”<br /> );<br /> $supports = array(‘title’, ‘editor’, ‘custom-fields’, ‘revisions’, ‘excerpt’);<br /> register_post_type( ‘event’,<br /> array(<br /> ‘labels’ =&gt; $labels,<br /> ‘public’ =&gt; true,<br /> ‘supports’ =&gt; $supports<br /> )<br /> );</p>\
<p>}</p>\
</blockquote>\
<p><strong>add_action</strong>告诉wordpress初始化时调用函数<strong>create_events</strong></p>\
<p><strong>$label</strong>数组告诉wordpress如何显示这个文章类型的相关信息</p>\
<p><strong>$supports</strong>数组告诉wordpress这个文章类型可以支持什么（比如文章摘要excerpt）</p>\
<p><strong>register_post_type</strong>就是在wordpress注册这个新的文章类型，这个函数有很多选项可用，具体可以参阅相关文章。</p>\
<p>一旦我们把代码添加到function.php后，我们可以在后台管理页面看到如下页面：</p>\

<h4>3.显示自定义文章类型</h4>\
<p>添加完事件之后发布，然后编辑事件，点击View Event（查看事件），你就可以看到事件就像正常文章一样显示了。</p>\
<p><img title="06-04_wordpress_custom_post_type_just_like_post" alt="wordpress-自定义文章类型" src="http://www.wpyou.com/wp-content/uploads/2011/07/0923327VC.jpg" width="550" height="451" /></p>\
<p>当 然我们在wordpress自定义文章类型是因为它们和一般的文章是不一样的。wordpress内部有一个很好的机制用来自定义文章类型的外观，如果你 曾经做过wordpress主题，应该知道，一般文章（post）对应的模板文件是single.php。wordpress3.0允许你用自定义的模板 文件（如single-event.php）来定制我们新的文章类型的外观。</p>\
<p>为了简便起见，我们将single.php复制并命名为single-event.php，为了演示看是否生效，我们对代码做一些改动。</p>\
<p><strong>原代码</strong>：</p>\
<blockquote>\
<p><code>&lt;</code><code>h1</code> <code>class</code><code>=</code><code>"entry-title"</code><code>&gt;&lt;?</code><code>php</code> <code>the_title(); ?&gt;&lt;/</code><code>h1</code><code>&gt;</code></p>\

</blockquote>\

<h4>4.列出自定义文章类型</h4>\
<p>前面我们讲解如何创建、显示单个事件，但是如果我们需要像博客日志一样列出所有的事件应该如何操作？</p>\
<p>这里有很多方法可以实现这个功能，一个通常使用的方法是通过<a href="http://codex.wordpress.org/Pages#Page_Templates">Page Template</a>（页面模板）来实现，这个方法如果使用几次后发现它并不难，只是需要的步骤多一点。</p>\
<p>（1）首先复制一份page.php并将它命名为page-events.php。（可以在wordpress创建很多不同页面模板）。</p>\
<p>（2）在page-events.php头部添加一下代码，这些代码告诉wordpress这是一个模板文件，以及这个模板文件的 相关信息。</p>\
<blockquote>\
<p>&lt;?php<br /> /**<br /> * Template Name: Events Template<br /> * @package WordPress<br /> * @subpackage Twenty_Ten<br /> * @since Twenty Ten 1.0<br /> */<br /> get_header(); ?&gt;</p>\
</blockquote>\
<p>（3）先用下面的代码来获取所有的事件文章.</p>\
<blockquote>\
<p><code>&lt;?php query_posts(</code><code>array</code><code>(</code><code>'post_type'</code><code>=&gt;</code><code>'event'</code><code>)); ?&gt;</code></p>\
</blockquote>\
<p>（4）然后用wordpress loop的方法来显示出所有的事件文章。</p>\
<blockquote>\
<p><code>&lt;?php </code><code>if</code> <code>( have_posts() ) </code><code>while</code> <code>( have_posts() ) : the_post(); ?&gt;</code></p>\

<p>这 样我们就创建好了一个自定义的页面模板，接着创建一个Event页面来显示这些事件，进入wordpress后台的页面部分，创建一个新页面命名为 Event，然后在页面属性的模板选项里，选择我们刚才创建的模板文件即可。发布完后，到网站查看这个页面，可以发现所有的事件文章都列在上面了。根据不 同的主题，你可能需要修改导航栏的相关代码。</p>\

<p> <p>[gallery link="file" ids="148,147,136"]</p>\
<p> <b>四八年写八零年冬再记绿野</b></p>\
<p align="left">岭南画派，敢为天下先，飞机轮船，举目所见者皆可入画。其创始者高剑父等人早在1920年代就提出“洋为中用”口号，比徐悲鸿要早。作为高剑父的关门弟子，主攻花鸟的叶绿野先生虽被老师认定“笔性文静”，却也一生高蹈，不涉颓靡</p>\
<p>“天地之大，什么样的鸟没有？只要能想象出来就够胆画！”岭南画派画家叶绿野认为自己真正进入绘画上的自由王国，还是在70岁以后。“过去画鸟雀还分辨一下种类，现在几乎是随心所欲，下笔毫不犹豫。”84岁的叶绿野，讲话糯糯地带着客家口音。</p>\
<p align="left">叶绿野这个名字，由岭南画派创始人高剑父代起，取自《牡丹亭》“月明无犬吠黄花，雨过有人耕绿野”的意境。高剑父说，叶绿野“笔性文静，行笔流畅淡雅”，主张他专攻花鸟。</p>\
<p align="left">虽然一早由老师高剑父定下花鸟画的路子，但叶绿野觉得自己年轻时画画，总免不了小心谨慎；年岁大了后，反而更加收放自如，颜色的对比也越发大胆，敢画“万绿丛中一点红”；画红色的荔枝，也要先铺上一层绿。</p>\
<p align="left">叶绿野觉得，一直画到1980年代，自己才堪配“画家”之称：“以前是挂在老师的‘户口’上，后来才慢慢把‘户口’独立出来。”</p>\
<p align="left">叶绿野认为自己的画特点是“清新润泽”，“干净明丽，生气盎然”。他认为自己的感觉至今还没走下坡路，画中还是充满生命力：“残山剩水之类我从来不画———那不是我。”</p>\
<p align="left">为避免干扰，84岁的叶绿野现在仍然每天凌晨4点起来画画，天明方歇。即使站到脚都麻木了，画完后也一定要把画笔和用具洗干净，地上的铅笔屑也要扫得干干净净。</p>\
<p align="left">美术评论家李伟铭认为：“如果说，高剑父擅长画鸷禽猛兽，以此来抒发壮怀激烈的‘革命’情感的话，那么，叶先生则钟情于弱小生灵，乐于从鸟雀跳踯的节奏和吱喳鸣啭之音中，倾听天籁的启示。当然，这种趣味的区别既描述了时代情境的变迁，同时也凸现了艺术家天性的差异。”</p>\
<p align="left"><b>“人穷猪命短”</b></p>\
<p>广东东北龙川县有个才子村，叶氏曾是当地富甲一方的大家族。叶绿野的父亲叶含华，祖上本有40多亩地，但为了3个儿子读中学、念大学，10年间花光了所有积蓄，变卖田地，乃至负债累累。</p>\
<p align="left">叶绿野记忆中，大哥画画悟性最好，总能拿到各种绘画奖项。老家书房的墙上曾经醒目地挂着大哥得的一幅红绸镶边的奖状，上面写着“美不胜收”。然而大哥风流倜傥，散钱阔绰，在广州市立美术学校读书的他，一年要花费600银元，而一般的同学一年100元就足够了。</p>\
<p align="left">叶绿野记得，有一次大哥从外地写信回来，说急需用钱。其时家里已经一贫如洗。但父亲还是把一头70斤的瘦猪卖了。过了很久，父亲才叹了口气，说，“真是人穷猪命短啊。”又嘱咐叶绿野，“人穷志不穷，将来才能做大事。”</p>\
<p align="left">除 了两位兄长，叶绿野走上绘画之路还跟一位堂兄有关。堂兄毕业于刘海粟办的上海新华艺专，毕业后在广东省立老隆师范学校教美术。叶绿野到这所学校念书，便是 这位堂兄垫的学费；学校办图画比赛，叶绿野的画第一次拿奖，也是这位堂兄给他买的画纸和颜料。叶绿野当时画了一幅风景水彩和一幅漫画《自寻死路》———一 个人头钻进骷髅头里———讽刺汪精卫。</p>\
<p align="left">之后，叶绿野在学校就出了名。同学、老师纷纷找他要画。“人家买纸来，能不画吗？”毕业后也因为画画出名，叶绿野在学校的附小教起了美术。</p>\
<p>1946年，因为不愿意一辈子都教小学生画画，叶绿野辞去小学教职后，去广州考取了南中美术学院。南中美院是由高剑父在同年创办。学校本来是9月2日开学，但因为家穷，直到9月中旬，叶绿野还没收到家里寄来的学费。他心中焦急又无技可施，便索性先去学校看看情况。他记得南中美院是座古色古香的院子，圆拱门、玻璃窗，还有半截红墙。他去教务处说了迟迟未报到的原因，处长便让他去后花园找高院长。</p>\
<p align="left">叶绿野见到了高剑父，讲明了情况。高剑父马上让叶绿野直接去上课，免了他的学费。</p>\
<p align="left">1947年，叶绿野与其他4个学生一起，被高剑父选入春睡画院，那时，1923年由高剑文一手创办的春睡画院已不公开招生。高剑父还让他兼职广州市立艺专的财务出纳，借以维持生活。</p>\
<p align="left"><b>“我是杂种，你们就是老杂种”</b></p>\
<p align="left">对高剑父，叶绿野的印象是“很胆量”。高剑父小时候家里也很穷，后来东渡日本，在东京美术学院学习绘画，之后参加“白马画会”等艺术团体。没得吃，就住到廖仲恺家里，教何香凝(廖仲恺夫人)画画。在他家里，高剑父认识了孙中山，参加了革命。之后孙中山派高剑父回广东组织同盟会，高剑父任同盟会会长。后来的黄花岗起义，高剑父正是发起人之一。辛亥革命时，高剑父担任“支那暗杀团”团长，带着20多个敢死队员，成功刺杀了当时的广东都督凤山。辛亥革命后，高剑父脱离政界，倾心于艺术。</p>\
<p align="left">除了社会革命外，高剑父亦在国画界发起革命。从日本回来后，高剑父便大力提倡“美术革命”、“国画革命”。1921年， 高剑父与高奇峰、陈树人等一起倡导“新国画”，并举办“第一次美术展览会”，展出了许多在思想内容、题材风格上与过去迥然不同的新国画。高剑父提出口号， “高举艺术革命的大纛，从广东发难起来”。在《我的现代国画观》中，高剑父提出：“不管什么画派，也不分中国和外国，都以去芜存菁，一炉共冶，创立出一种 不脱离民族风格而又推陈出新的现代新国画。”</p>\
<p>1923年，高剑父创办“春睡画院”，公开招收学生，推广“新国画”主张。之后，又陆续创办南中美术专科学校、广州市立美术专科学校，并先后任广东美术学会会长、中央大学及中山大学教授。</p>\
<p align="left">因高剑父、高奇峰、陈树人等皆是广东人，上海国画家们率先称他们为“岭南画派”，而“二高一陈”遂被视为岭南画派创始人，被后人称为“岭南三杰”。被视为岭南画派第二代的方人定、黎雄才、关山月等便是在春睡画院亲炙高剑父“中西结合”主张的。</p>\
<p align="left">高 剑父主张国画应反映现实，飞机、汽车皆可入画；主张国画中西结合，洋为中用，将油画、水彩的技法引入国画。但传统派极力反对，派流氓去扰乱高剑父师生的画 展，还骂高剑父为“杂种”。高剑父回之曰，如果我是杂种的话，那你们就是老杂种。你们不是沿用佛教传入中国后变革的国画画法么？你们老杂种就可以，我再杂 一下怎么就不行？</p>\
<p align="left">“以前画山水，必须画茅棚，都是山上一个茅棚、水上一个小亭子、有个老人驼着背在那里撑船。”而高剑父看到什么就画什么，叶绿野将高剑父在题材上的革新类比为孙中山的“剪辫革命”。</p>\
<p align="left">虽然高剑父很爱护学生，但一发起脾气来也凶得不得了。叶绿野记得高剑父跟他们发过两次火。一次是叶绿野画了一张孔雀，高剑父觉得不错，便让他找其他老师帮忙补后景。结果补得黑了一点。高剑父看得冒火，责问是谁补的，在房间里走来走去，脸色发青。</p>\
<p align="left">在叶绿野看来，中国画最大的改革就出自岭南画派。国画的“中西结合”改革并不是如一般人所说是徐悲鸿的功劳，高剑父等早在1920年代就提出“洋为中用”口号，比徐悲鸿要早。</p>\
<p>只是到现在，全国各种画派都讲求“中西结合”，岭南画派的优势就没有特别凸显了。“我们宣传得不够，没有自身的理论队伍，也没人像徐悲鸿那样在中央美术学院做院长。”叶绿野认为，地缘政治也是影响岭南画派声誉的因素，“如果国都移到广州来，我们会是这个样吗？”</p>\
<p align="left">高剑父之后，方人定、黄少强、赵少昂、黎雄才、关山月、杨善深等成为岭南画派第二代传人，自1930年代崭露头角，影响一直延续当代。其中赵少昂、黎雄才、关山月、杨善深4人被称作当代岭南画派四大画家。再之后，一般认为岭南画派又有杨之光、林墉、陈金章等人。</p>\
<p align="left">美术界有人认为岭南画派没有形成比较连贯的、明确的艺术主张和艺术风格，“岭南画派式微”。</p>\
<p align="left">而在叶绿野看来，凡承继过岭南画派“中西结合”风格的，皆可以算作岭南画派的画家：“以前我们就十来个人，春睡画院也就一百几十个，现在全国有上万、几十万了。这不是壮大了么？”</p>\
<p align="left"><b>当老师后还做学生</b></p>\
<p align="left">1949年广州解放前，高剑父已经移居澳门，叶绿野在广州举目无亲，便随着惶惶人群流落到香港，碰到了关山月。其时关山月正因从事共产党地下活动被国民党通缉而流亡香港，与一帮广州画家一起，成立了“广州美术工作者留港协会”，叶绿野遂“归队”。</p>\
<p align="left">广州解放那天，巨幅毛泽东肖像油画《中国人民站起来了》挂在了当时广州最高的爱群大厦外面，从11楼垂到2楼。这幅画便是留港画家们所为。关山月作组织和指导，北京来的画家们画毛泽东的脸部，杨秋人画毛泽东的衣服。叶绿野也参与其中，画了毛泽东的一粒纽扣：“别看一粒纽扣，足有一平方米大！”</p>\
<p>在“反右”及“文革”中，叶绿野的这段香港经历也成为一项罪状，被认作“美蒋特务”。而叶绿野在南中美院读书时担任出纳之职，也成为他是“牛鬼蛇神”的证明。叶绿野本来在广东西江文工团担任美术队长，一下子被发配到增城中学教书。</p>\
<p align="left">“说来好笑，那年月养鸟无罪，画鸟有罪；打麻雀是有益运动，画雀却属被禁之列。”叶绿野一幅画中有只鸟飞往左边，也被批判是想飞向西方。因为按照地图方位规则，左边是西方。</p>\
<p align="left">不能画花鸟，叶绿野改弦更张，一本正经教起生物来。他还给学校画了200多幅鸟类标本教学挂图。</p>\
<p align="left">“我学过《生物学》，知道天下有30多万种鸟。”在增城中学，叶绿野还学了《教育学》、《人体解剖》，又藉着教生物课，接触了巴甫洛夫等各种学说，对人体和鸟类解剖非常熟悉，对生物的生活规律也都了如指掌。在叶绿野看来，“这其实是画家的基本功”。</p>\
<p align="left">叶绿野曾读过4遍《艺海拾贝》，这还得“感谢”“文革”时对秦牧的批判：“秦牧对人生的思考是很有深度和厚度的，在大灾难面前，有秦牧那样的定力不容易。如果你竟然能够像钱锺书先生那样‘浑然不觉’，像华君武先生那样及时幽上一默，那你就大大超越灾难本身了。”</p>\
<p align="left">“文 革”结束后，广州美术学院恢复招生，关山月在那里担任副院长。由于师资缺乏，就想把叶绿野调过去。但增城中学舍不得放人。叶绿野就听从朋友建议，给省委写 了封信说明原委。那时吴南生正担任广东省副省长，兼科教办主任，掌管大学，人称“铁包公”，看了信后马上就批复，要把叶绿野调过来。但广东画院也想要叶绿 野，跟广州美术学院争。吴南生就干脆先把叶绿野调到省委。“结果我接到通知去省委报到。之后两三天，我才被通知去广州美院。”叶绿野一生感激的两个人，除 了恩师高剑父，就是这位吴南生。</p>\
<p>忆及当年，85岁的吴南生先生说：“假如叶绿野被埋没，作为主管文化的领导，我就对不起岭南画派。叶绿野是高剑父的关门弟子，他的画作闪耀着岭南画派的光辉。”</p>\
<p align="left">在广州美院，叶绿野秉承岭南画派的特点，带领学生四处写生。在他看来，写生能使人保持对事物的新鲜感，能慢慢发现世界每一类物种的多样性。</p>\
<p align="left">叶 绿野还在美院当过“学生”。有天晚上，他发现在广州美院读书的女儿在翻阅讲义《平面构成学》。这一课在当年的南中美院并没教过。叶绿野就跟着女儿一起去听 课，认真记笔记，课后还跟女儿的老师交流。平面构成理论特别强调对比，而国画根本就是一种对比的艺术———疏密、冷暖、大小。之后，叶绿野有意无意把这种 理论运用在画中，画风不知不觉又为之一变。</p>\
<p align="left">2006年12月28日，叶绿野美术馆在其家乡广东河源市建成开馆，展出叶绿野捐赠的100幅国画作品。叶绿野一直主张绘画艺术最终要“示众”，但“必须示人以弦外之音、画外之意———如果我的画不能给人以画外的联想，那是一种悲哀，也是画者的一种失败”。(本报记者　万静)</p>\

<p> <b>四八年写八零年冬再记绿野</b></p>\
<p align="left">岭南画派，敢为天下先，飞机轮船，举目所见者皆可入画。其创始者高剑父等人早在1920年代就提出“洋为中用”口号，比徐悲鸿要早。作为高剑父的关门弟子，主攻花鸟的叶绿野先生虽被老师认定“笔性文静”，却也一生高蹈，不涉颓靡</p>\
<p>“天地之大，什么样的鸟没有？只要能想象出来就够胆画！”岭南画派画家叶绿野认为自己真正进入绘画上的自由王国，还是在70岁以后。“过去画鸟雀还分辨一下种类，现在几乎是随心所欲，下笔毫不犹豫。”84岁的叶绿野，讲话糯糯地带着客家口音。</p>\
<p align="left">叶绿野这个名字，由岭南画派创始人高剑父代起，取自《牡丹亭》“月明无犬吠黄花，雨过有人耕绿野”的意境。高剑父说，叶绿野“笔性文静，行笔流畅淡雅”，主张他专攻花鸟。</p>\
<p align="left">虽然一早由老师高剑父定下花鸟画的路子，但叶绿野觉得自己年轻时画画，总免不了小心谨慎；年岁大了后，反而更加收放自如，颜色的对比也越发大胆，敢画“万绿丛中一点红”；画红色的荔枝，也要先铺上一层绿。</p>\
<p align="left">叶绿野觉得，一直画到1980年代，自己才堪配“画家”之称：“以前是挂在老师的‘户口’上，后来才慢慢把‘户口’独立出来。”</p>\
<p align="left">叶绿野认为自己的画特点是“清新润泽”，“干净明丽，生气盎然”。他认为自己的感觉至今还没走下坡路，画中还是充满生命力：“残山剩水之类我从来不画———那不是我。”</p>\
<p align="left">为避免干扰，84岁的叶绿野现在仍然每天凌晨4点起来画画，天明方歇。即使站到脚都麻木了，画完后也一定要把画笔和用具洗干净，地上的铅笔屑也要扫得干干净净。</p>\
<p align="left">美术评论家李伟铭认为：“如果说，高剑父擅长画鸷禽猛兽，以此来抒发壮怀激烈的‘革命’情感的话，那么，叶先生则钟情于弱小生灵，乐于从鸟雀跳踯的节奏和吱喳鸣啭之音中，倾听天籁的启示。当然，这种趣味的区别既描述了时代情境的变迁，同时也凸现了艺术家天性的差异。”</p>\
<p align="left"><b>“人穷猪命短”</b></p>\
<p>广东东北龙川县有个才子村，叶氏曾是当地富甲一方的大家族。叶绿野的父亲叶含华，祖上本有40多亩地，但为了3个儿子读中学、念大学，10年间花光了所有积蓄，变卖田地，乃至负债累累。</p>\
<p align="left">叶绿野记忆中，大哥画画悟性最好，总能拿到各种绘画奖项。老家书房的墙上曾经醒目地挂着大哥得的一幅红绸镶边的奖状，上面写着“美不胜收”。然而大哥风流倜傥，散钱阔绰，在广州市立美术学校读书的他，一年要花费600银元，而一般的同学一年100元就足够了。</p>\
<p align="left">叶绿野记得，有一次大哥从外地写信回来，说急需用钱。其时家里已经一贫如洗。但父亲还是把一头70斤的瘦猪卖了。过了很久，父亲才叹了口气，说，“真是人穷猪命短啊。”又嘱咐叶绿野，“人穷志不穷，将来才能做大事。”</p>\
<p align="left">除 了两位兄长，叶绿野走上绘画之路还跟一位堂兄有关。堂兄毕业于刘海粟办的上海新华艺专，毕业后在广东省立老隆师范学校教美术。叶绿野到这所学校念书，便是 这位堂兄垫的学费；学校办图画比赛，叶绿野的画第一次拿奖，也是这位堂兄给他买的画纸和颜料。叶绿野当时画了一幅风景水彩和一幅漫画《自寻死路》———一 个人头钻进骷髅头里———讽刺汪精卫。</p>\
<p align="left">之后，叶绿野在学校就出了名。同学、老师纷纷找他要画。“人家买纸来，能不画吗？”毕业后也因为画画出名，叶绿野在学校的附小教起了美术。</p>\
<p>1946年，因为不愿意一辈子都教小学生画画，叶绿野辞去小学教职后，去广州考取了南中美术学院。南中美院是由高剑父在同年创办。学校本来是9月2日开学，但因为家穷，直到9月中旬，叶绿野还没收到家里寄来的学费。他心中焦急又无技可施，便索性先去学校看看情况。他记得南中美院是座古色古香的院子，圆拱门、玻璃窗，还有半截红墙。他去教务处说了迟迟未报到的原因，处长便让他去后花园找高院长。</p>\
<p align="left">叶绿野见到了高剑父，讲明了情况。高剑父马上让叶绿野直接去上课，免了他的学费。</p>\
<p align="left">1947年，叶绿野与其他4个学生一起，被高剑父选入春睡画院，那时，1923年由高剑文一手创办的春睡画院已不公开招生。高剑父还让他兼职广州市立艺专的财务出纳，借以维持生活。</p>\
<p align="left"><b>“我是杂种，你们就是老杂种”</b></p>\
<p align="left">对高剑父，叶绿野的印象是“很胆量”。高剑父小时候家里也很穷，后来东渡日本，在东京美术学院学习绘画，之后参加“白马画会”等艺术团体。没得吃，就住到廖仲恺家里，教何香凝(廖仲恺夫人)画画。在他家里，高剑父认识了孙中山，参加了革命。之后孙中山派高剑父回广东组织同盟会，高剑父任同盟会会长。后来的黄花岗起义，高剑父正是发起人之一。辛亥革命时，高剑父担任“支那暗杀团”团长，带着20多个敢死队员，成功刺杀了当时的广东都督凤山。辛亥革命后，高剑父脱离政界，倾心于艺术。</p>\
<p align="left">除了社会革命外，高剑父亦在国画界发起革命。从日本回来后，高剑父便大力提倡“美术革命”、“国画革命”。1921年， 高剑父与高奇峰、陈树人等一起倡导“新国画”，并举办“第一次美术展览会”，展出了许多在思想内容、题材风格上与过去迥然不同的新国画。高剑父提出口号， “高举艺术革命的大纛，从广东发难起来”。在《我的现代国画观》中，高剑父提出：“不管什么画派，也不分中国和外国，都以去芜存菁，一炉共冶，创立出一种 不脱离民族风格而又推陈出新的现代新国画。”</p>\
<p>1923年，高剑父创办“春睡画院”，公开招收学生，推广“新国画”主张。之后，又陆续创办南中美术专科学校、广州市立美术专科学校，并先后任广东美术学会会长、中央大学及中山大学教授。</p>\
<p align="left">因高剑父、高奇峰、陈树人等皆是广东人，上海国画家们率先称他们为“岭南画派”，而“二高一陈”遂被视为岭南画派创始人，被后人称为“岭南三杰”。被视为岭南画派第二代的方人定、黎雄才、关山月等便是在春睡画院亲炙高剑父“中西结合”主张的。</p>\
<p align="left">高 剑父主张国画应反映现实，飞机、汽车皆可入画；主张国画中西结合，洋为中用，将油画、水彩的技法引入国画。但传统派极力反对，派流氓去扰乱高剑父师生的画 展，还骂高剑父为“杂种”。高剑父回之曰，如果我是杂种的话，那你们就是老杂种。你们不是沿用佛教传入中国后变革的国画画法么？你们老杂种就可以，我再杂 一下怎么就不行？</p>\
<p align="left">“以前画山水，必须画茅棚，都是山上一个茅棚、水上一个小亭子、有个老人驼着背在那里撑船。”而高剑父看到什么就画什么，叶绿野将高剑父在题材上的革新类比为孙中山的“剪辫革命”。</p>\
<p align="left">虽然高剑父很爱护学生，但一发起脾气来也凶得不得了。叶绿野记得高剑父跟他们发过两次火。一次是叶绿野画了一张孔雀，高剑父觉得不错，便让他找其他老师帮忙补后景。结果补得黑了一点。高剑父看得冒火，责问是谁补的，在房间里走来走去，脸色发青。</p>\
<p align="left">在叶绿野看来，中国画最大的改革就出自岭南画派。国画的“中西结合”改革并不是如一般人所说是徐悲鸿的功劳，高剑父等早在1920年代就提出“洋为中用”口号，比徐悲鸿要早。</p>\
<p>只是到现在，全国各种画派都讲求“中西结合”，岭南画派的优势就没有特别凸显了。“我们宣传得不够，没有自身的理论队伍，也没人像徐悲鸿那样在中央美术学院做院长。”叶绿野认为，地缘政治也是影响岭南画派声誉的因素，“如果国都移到广州来，我们会是这个样吗？”</p>\
<p align="left">高剑父之后，方人定、黄少强、赵少昂、黎雄才、关山月、杨善深等成为岭南画派第二代传人，自1930年代崭露头角，影响一直延续当代。其中赵少昂、黎雄才、关山月、杨善深4人被称作当代岭南画派四大画家。再之后，一般认为岭南画派又有杨之光、林墉、陈金章等人。</p>\
<p align="left">美术界有人认为岭南画派没有形成比较连贯的、明确的艺术主张和艺术风格，“岭南画派式微”。</p>\
<p align="left">而在叶绿野看来，凡承继过岭南画派“中西结合”风格的，皆可以算作岭南画派的画家：“以前我们就十来个人，春睡画院也就一百几十个，现在全国有上万、几十万了。这不是壮大了么？”</p>\
<p align="left"><b>当老师后还做学生</b></p>\
<p align="left">1949年广州解放前，高剑父已经移居澳门，叶绿野在广州举目无亲，便随着惶惶人群流落到香港，碰到了关山月。其时关山月正因从事共产党地下活动被国民党通缉而流亡香港，与一帮广州画家一起，成立了“广州美术工作者留港协会”，叶绿野遂“归队”。</p>\
<p align="left">广州解放那天，巨幅毛泽东肖像油画《中国人民站起来了》挂在了当时广州最高的爱群大厦外面，从11楼垂到2楼。这幅画便是留港画家们所为。关山月作组织和指导，北京来的画家们画毛泽东的脸部，杨秋人画毛泽东的衣服。叶绿野也参与其中，画了毛泽东的一粒纽扣：“别看一粒纽扣，足有一平方米大！”</p>\
<p>在“反右”及“文革”中，叶绿野的这段香港经历也成为一项罪状，被认作“美蒋特务”。而叶绿野在南中美院读书时担任出纳之职，也成为他是“牛鬼蛇神”的证明。叶绿野本来在广东西江文工团担任美术队长，一下子被发配到增城中学教书。</p>\
<p align="left">“说来好笑，那年月养鸟无罪，画鸟有罪；打麻雀是有益运动，画雀却属被禁之列。”叶绿野一幅画中有只鸟飞往左边，也被批判是想飞向西方。因为按照地图方位规则，左边是西方。</p>\
<p align="left">不能画花鸟，叶绿野改弦更张，一本正经教起生物来。他还给学校画了200多幅鸟类标本教学挂图。</p>\
<p align="left">“我学过《生物学》，知道天下有30多万种鸟。”在增城中学，叶绿野还学了《教育学》、《人体解剖》，又藉着教生物课，接触了巴甫洛夫等各种学说，对人体和鸟类解剖非常熟悉，对生物的生活规律也都了如指掌。在叶绿野看来，“这其实是画家的基本功”。</p>\
<p align="left">叶绿野曾读过4遍《艺海拾贝》，这还得“感谢”“文革”时对秦牧的批判：“秦牧对人生的思考是很有深度和厚度的，在大灾难面前，有秦牧那样的定力不容易。如果你竟然能够像钱锺书先生那样‘浑然不觉’，像华君武先生那样及时幽上一默，那你就大大超越灾难本身了。”</p>\
<p align="left">“文 革”结束后，广州美术学院恢复招生，关山月在那里担任副院长。由于师资缺乏，就想把叶绿野调过去。但增城中学舍不得放人。叶绿野就听从朋友建议，给省委写 了封信说明原委。那时吴南生正担任广东省副省长，兼科教办主任，掌管大学，人称“铁包公”，看了信后马上就批复，要把叶绿野调过来。但广东画院也想要叶绿 野，跟广州美术学院争。吴南生就干脆先把叶绿野调到省委。“结果我接到通知去省委报到。之后两三天，我才被通知去广州美院。”叶绿野一生感激的两个人，除 了恩师高剑父，就是这位吴南生。</p>\
<p>忆及当年，85岁的吴南生先生说：“假如叶绿野被埋没，作为主管文化的领导，我就对不起岭南画派。叶绿野是高剑父的关门弟子，他的画作闪耀着岭南画派的光辉。”</p>\
<p align="left">在广州美院，叶绿野秉承岭南画派的特点，带领学生四处写生。在他看来，写生能使人保持对事物的新鲜感，能慢慢发现世界每一类物种的多样性。</p>\
<p align="left">叶 绿野还在美院当过“学生”。有天晚上，他发现在广州美院读书的女儿在翻阅讲义《平面构成学》。这一课在当年的南中美院并没教过。叶绿野就跟着女儿一起去听 课，认真记笔记，课后还跟女儿的老师交流。平面构成理论特别强调对比，而国画根本就是一种对比的艺术———疏密、冷暖、大小。之后，叶绿野有意无意把这种 理论运用在画中，画风不知不觉又为之一变。</p>\
<p align="left">2006年12月28日，叶绿野美术馆在其家乡广东河源市建成开馆，展出叶绿野捐赠的100幅国画作品。叶绿野一直主张绘画艺术最终要“示众”，但“必须示人以弦外之音、画外之意———如果我的画不能给人以画外的联想，那是一种悲哀，也是画者的一种失败”。(本报记者　万静)</p>\
